# St. Charles' College Web App

## Overview

This project is the official website for St. Charles' College, Onitsha. It's built to give students, parents, alumni, and prospective members a comprehensive and engaging digital experience. Think of it as a central hub where you can explore everything about the "Varsity on the Niger" – from its rich history and academic programs to admissions information, campus news, and project initiatives. It solves the problem of needing a modern, accessible, and informative online presence for a prestigious educational institution, making it easy for anyone to connect with the college's legacy and future.

## Features

- **Comprehensive Information Pages**: Dedicated sections for About Us, Academics, Admissions, News & Updates, Gallery, Projects, and Contact.
- **Dynamic Hero Section**: Engaging hero banner with rotating content to highlight key aspects of the college.
- **Interactive History Timeline**: Visually presents the college's journey through different historical eras.
- **Detailed Curriculum Overview**: Showcases academic departments, courses, and examination preparation methods.
- **Admissions Guide**: Clear, step-by-step instructions for the application process, including requirements and FAQs.
- **Image Gallery with Filtering**: Explore campus life, events, and academic achievements through categorized photos.
- **Development Projects Showcase**: Highlights ongoing and completed initiatives, with options for community involvement.
- **Responsive Navigation**: Mobile-friendly side drawer menu for seamless browsing on any device.
- **Theme Toggle**: Easily switch between light and dark modes for a personalized viewing experience.
- **Smooth Animations**: Utilizes Framer Motion for subtle, elegant transitions and interactive elements.
- **Scroll-to-Top Button**: Convenient navigation back to the top of long pages.
- **SEO Optimized**: Metadata included for improved search engine visibility.

## Getting Started

To get this project up and running on your local machine, follow these steps:

### Installation

1.  **Clone the Repository**:
    ```bash
    git clone <repository-url>
    ```
    (Replace `<repository-url>` with the actual URL of your repository.)
2.  **Navigate to the project directory**:
    ```bash
    cd st-charles-college-webapp
    ```
3.  **Install dependencies**:
    Using npm:
    ```bash
    npm install
    ```
    Or using yarn:
    ```bash
    yarn install
    ```

### Environment Variables

This project does not require any custom environment variables for basic development or production deployment as it primarily relies on static content and Next.js's built-in configurations.

## Usage

Once you've installed the dependencies, you can run the development server or build the project for production.

### Running the Development Server

To start the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the application. The page will auto-update as you make changes to the source code.

### Building for Production

To create an optimized production build:

```bash
npm run build
# or
yarn build
```

This command will compile and optimize the application for deployment.

### Starting the Production Server

To run the built application in a production environment:

```bash
npm run start
# or
yarn start
```

This will serve the optimized build, typically at `http://localhost:3000`.
